# ascending-primes-scala

Задача из Rosetta Code: найти все простые числа, в которых десятичные цифры строго возрастают (например, 1379).

### Описание

Программа генерирует все числа с возрастающими десятичными цифрами (например, 1, 12, 123, 135, 1479 и т.д.). Затем она фильтрует только те числа, которые являются простыми.

### Как запустить

Для запуска проекта вам необходимо:

1. Установить Scala CLI и AdoptOpenJDK 8/11. Сделать это можно по инструкции:
https://docs.scala-lang.org/getting-started/install-scala.html (там есть все ссылки).
2. Клонировать проект или скачать исходники.
3. В командной строке сначала указать путь к папке, в которой находится проект, в моем случае так:
cd C:\Users\user\ascending-primes
Теперь осталось запустить программу! Вводим следом в командной строке:
scala-cli run .
